## 三、分步骤运行

### 3.1 环境验证

```bash
# 确认 Python 环境
/home/ditx/llm/voice/3D-Speaker/.3ds/bin/python3 --version

# 确认 GPU
nvidia-smi

# 确认模型路径
ls /data/models/Voice/
```

### 3.2 数据生成

```bash
cd /data/wbt333/DiTX-Clerk
/home/ditx/llm/voice/3D-Speaker/.3ds/bin/python3 prepare_data.py \
    --output_dir data --num_sft 100 --num_grpo 100
```

### 3.3 RAG 建库

```bash
/home/ditx/llm/voice/3D-Speaker/.3ds/bin/python3 setup_rag_db.py \
    --storage_dir ./rag_storage \
    --data_dir data/meeting_records \
    --skip_test
```

### 3.4 Pipeline 运行

```bash
# 基础模式（文本输出）
/home/ditx/llm/voice/3D-Speaker/.3ds/bin/python3 scripts/offline_pipeline.py \
    --audio examples/2speakers_example.wav \
    --output-dir output_test

# QA 模式（意图识别路由）
/home/ditx/llm/voice/3D-Speaker/.3ds/bin/python3 scripts/offline_pipeline_workflow.py \
    --audio examples/2speakers_example.wav \
    --enable-rag --mode qa \
    --query "这场会上张三说了什么"

# 总结模式 + Agentic RAG
/home/ditx/llm/voice/3D-Speaker/.3ds/bin/python3 scripts/offline_pipeline_workflow.py \
    --audio examples/2speakers_example.wav \
    --enable-rag --agentic-rag \
    --time-range 2026-05-01,2026-06-03
```

### 3.5 训练

```bash
# SFT 训练
/home/ditx/llm/voice/3D-Speaker/.3ds/bin/python3 train_sft.py \
    --model_name_or_path /data/models/Qwen/Qwen2.5-0.5B-Instruct \
    --train_data data/sft_train_data.jsonl \
    --output_dir saves/sft_model

# GRPO 训练
/home/ditx/llm/voice/3D-Speaker/.3ds/bin/python3 train_grpo.py \
    --model_name_or_path saves/sft_model \
    --train_data data/grpo_train_data.jsonl \
    --output_dir saves/grpo_model
```
