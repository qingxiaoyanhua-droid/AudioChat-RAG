## 完整训练流程

### 步骤 1: 准备训练数据

```bash

    --output_dir data \
    --num_sft 1000 \
    --num_grpo 1000 \
    --num_preference 1000 \
```

### 步骤 3: SFT 监督微调训练

#### 单卡训练（调试用）

```bash
python train_sft.py \
    --model_name_or_path /data/models/Fun-Audio-Chat-8B \
    --train_data data/sft_train_data.jsonl \
    --output_dir saves/sft_model \
    --per_device_train_batch_size 2 \
    --gradient_accumulation_steps 4 \
    --num_train_epochs 3 \
    --learning_rate 1e-5 \
    --use_lora True \
    --lora_r 64 \
    --lora_alpha 16
```

#### 多卡训练（学校 A100）

```bash
# 8 卡并行
torchrun --nproc_per_node=8 train_sft.py \
    --model_name_or_path /data/models/Fun-Audio-Chat-8B \
    --train_data data/sft_train_data.jsonl \
    --output_dir saves/sft_model \
    --per_device_train_batch_size 2 \
    --gradient_accumulation_steps 4 \
    --num_train_epochs 3 \
    --learning_rate 1e-5 \
    --use_lora True \
    --lora_r 64 \
    --lora_alpha 16
```

**训练参数说明:**

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| batch_size | 2 | 每卡批次大小（根据显存调整） |
| gradient_accumulation | 4 | 梯度累积步数 |
| learning_rate | 1e-5 | SFT 学习率 |
| lora_r | 64 | LoRA rank（越大参数越多） |
| epochs | 3 | 训练轮数 |

**预计训练时间:**
- Fun-Audio-Chat-8B: 约 2-3 小时（8×A100）
- Qwen-7B: 约 1-2 小时（8×A100）

### 步骤 4: GRPO 强化学习训练

#### 单卡训练

```bash
python train_grpo.py \
    --model_name_or_path saves/sft_model \
    --train_data data/grpo_train_data.jsonl \
    --output_dir saves/grpo_model \
    --num_generations 4 \
    --per_device_train_batch_size 1 \
    --gradient_accumulation_steps 4 \
    --num_train_epochs 3 \
    --learning_rate 1e-6 \
    --kl_coef 0.1
```

#### 多卡训练

```bash
torchrun --nproc_per_node=4 train_grpo.py \
    --model_name_or_path saves/sft_model \
    --train_data data/grpo_train_data.jsonl \
    --output_dir saves/grpo_model \
    --num_generations 4 \
    --per_device_train_batch_size 2 \
    --gradient_accumulation_steps 4 \
    --num_train_epochs 3 \
    --learning_rate 1e-6 \
    --kl_coef 0.1
```

**训练参数说明:**

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| num_generations | 4 | Group size（每个 prompt 生成 4 个 response） |
| batch_size | 1-2 | GRPO 显存占用大，建议小批次 |
| learning_rate | 1e-6 | GRPO 学习率（比 SFT 小） |
| kl_coef | 0.1 | KL 惩罚系数 |
| epochs | 3 | 训练轮数 |

**预计训练时间:**
- Fun-Audio-Chat-8B: 约 4-6 小时（4×A100）
- Qwen-7B: 约 2-4 小时（4×A100）

## 常见问题

### Q1: 显存不足怎么办？

**解决方案:**
1. 减小 batch_size（2 → 1）
2. 增加 gradient_accumulation_steps
3. 使用 DeepSpeed ZeRO-2/3
4. 启用 gradient_checkpointing

```bash
# 修改训练参数
--per_device_train_batch_size 1
--gradient_accumulation_steps 8
--gradient_checkpointing True
```

### Q2: 训练 loss 不下降？

**可能原因:**
1. 学习率太小/太大
2. 数据质量问题
3. 模型过拟合

**解决方案:**
1. 调整学习率（1e-5 → 5e-5 或 1e-6）
2. 检查数据格式是否正确
3. 增加正则化（KL penalty）

### Q3: GRPO 训练不稳定？

**解决方案:**
1. 增加 KL 惩罚系数（0.1 → 0.2）
2. 减小 group_size（4 → 2）
3. 使用 warmup 调度
4. Gradient clipping（max_norm=1.0）
