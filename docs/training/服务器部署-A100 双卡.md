## 完整训练流程（2 卡 A100）

### 1. 准备训练数据

```bash
python3 prepare_data.py \
    --output_dir data \
    --num_sft 1000 \
    --num_grpo 1000 \
    --num_rag 50
```

### 2. 搭建 RAG 数据库

```bash
python3 setup_rag_db.py \
    --storage_dir ./rag_storage \
    --data_dir data/meeting_records
```

### 3. SFT 训练（2 卡并行）

```bash
# 2 卡 A100 训练
torchrun --nproc_per_node=2 train_sft.py \
    --model_name_or_path /data/models/Voice/FunAudioLLM/Fun-Audio-Chat-8B \
    --train_data data/sft_train_data.jsonl \
    --output_dir saves/sft_model \
    --per_device_train_batch_size 2 \
    --gradient_accumulation_steps 4 \
    --num_train_epochs 3 \
    --learning_rate 1e-5 \
    --use_lora True \
    --lora_r 64 \
    --lora_alpha 16
```

**如果显存不足:**
```bash
torchrun --nproc_per_node=2 train_sft.py \
    --model_name_or_path /data/models/Voice/FunAudioLLM/Fun-Audio-Chat-8B \
    --train_data data/sft_train_data.jsonl \
    --output_dir saves/sft_model \
    --per_device_train_batch_size 1 \
    --gradient_accumulation_steps 8 \
    --gradient_checkpointing True \
    --num_train_epochs 3 \
    --learning_rate 1e-5 \
    --use_lora True
```

### 4. GRPO 训练（2 卡并行）

```bash
# 2 卡 A100 训练
torchrun --nproc_per_node=2 train_grpo.py \
    --model_name_or_path saves/sft_model \
    --train_data data/grpo_train_data.jsonl \
    --output_dir saves/grpo_model \
    --num_generations 4 \
    --per_device_train_batch_size 1 \
    --gradient_accumulation_steps 4 \
    --num_train_epochs 3 \
    --learning_rate 1e-6 \
    --kl_coef 0.1
```

## 显存优化配置

**A100 40GB 显存优化:**

| 配置 | batch_size | gradient_accumulation | 适用场景 |
|------|------------|----------------------|----------|
| 标准 | 2 | 4 | 显存充足 |
| 平衡 | 1 | 8 | 推荐配置 |
| 省显存 | 1 | 16 | 显存紧张 |

**额外优化选项:**
```bash
--gradient_checkpointing True    # 梯度检查点，节省 40% 显存
--fp16 True                      # FP16 混合精度
--optim adamw_apex_fused         # 优化器（如果可用）
```

## 常见问题

### Q1: torchrun 找不到

```bash
# 安装 torch.distributed
pip3 install torch torchelastic

# 或使用 python -m torch.distributed.launch
python3 -m torch.distributed.launch --nproc_per_node=2 train_sft.py ...
```

### Q2: 模型路径不存在

```bash
# 检查模型目录
ls /data/models/Voice/FunAudioLLM/

# 如果模型不存在，修改为实际路径或使用 HuggingFace 自动下载
--model_name_or_path funaudio/fun-audio-chat-8b
```

### Q3: 依赖冲突

```bash
# 创建虚拟环境（如果需要）
python3 -m venv ditx_env
source ditx_env/bin/activate

# 重新安装依赖
pip install -r requirements.txt
```

### Q4: CUDA 版本不匹配

```bash
# 安装匹配 CUDA 12.2 的 torch
pip3 install torch --index-url https://download.pytorch.org/whl/cu122
```
