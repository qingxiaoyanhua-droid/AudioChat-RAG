## 📊 完整训练流程

### SFT 训练

```bash
python3 train_sft.py \
    --model_name_or_path /home/wbt333/.cache/modelscope/hub/models/Qwen/Qwen2-0.5B-Instruct \
    --train_data data/sft_train_data.jsonl \
    --output_dir saves/sft_model_qwen05 \
    --per_device_train_batch_size 2 \
    --gradient_accumulation_steps 4 \
    --num_train_epochs 3 \
    --learning_rate 2e-4 \
    --use_lora True \
    --lora_r 32 \
    --lora_alpha 16 \
    --logging_steps 5 \
    --save_steps 50 \
    --fp16 True \
    2>&1 | tee logs/sft_train.log
```

### GRPO 训练

```bash
python3 train_grpo.py \
    --model_name_or_path saves/sft_model_qwen05 \
    --train_data data/grpo_train_data.jsonl \
    --output_dir saves/grpo_model_qwen05 \
    --num_generations 4 \
    --per_device_train_batch_size 1 \
    --gradient_accumulation_steps 4 \
    --num_train_epochs 2 \
    --learning_rate 1e-5 \
    --kl_coef 0.1 \
    --logging_steps 5 \
    --save_steps 30 \
    --fp16 True \
    2>&1 | tee logs/grpo_train.log
```
