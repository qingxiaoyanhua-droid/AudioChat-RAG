# 简历项目描述 - 可直接复制版

## 版本 A: 详细版（适合 2 页简历）

---

**2025.12 - 至今 &nbsp;&nbsp; 拓数派科技有限公司 &nbsp;&nbsp; 核心研发负责人**

**AudioChat-RAG - 多模态语音对话与大模型检索增强系统**

**项目概述**：设计实现基于本地大模型的语音转录与会议分析系统，搭建完整 Agent Skills 架构，落地多说话人会议全流程自动化处理方案。核心贡献为 RAG 检索增强与 GRPO 质量评估模块。

**核心工作**：
- **RAG 检索增强模块**：基于 ChromaDB 构建向量存储系统，使用 BGE-Large-ZH 实现中文语义检索，引入时间衰减重排序机制（半衰期 7 天），支持说话人过滤检索；**RAG 将回答准确性提升 31% (0.64 → 0.84)**，检索延迟 < 100ms
- **GRPO 质量评估模块**：设计多维度评估器（CER 分数 40% + 流畅度 30% + 相关性 30%），基于 CosyVoice GRPO 实现推理时评估，无需训练即可评估生成质量；**评估结果与人工标注相关性达 0.82**
- **端到端 Pipeline 集成**：搭建 FunASR（说话人分离+ASR）+ RAG + LLM（Fun-Audio-Chat-8B/Qwen3-30B）+ TTS（CosyVoice）五模块流水线，支持 S2T/S2S 双模式；**延迟从 23.5s 优化至 8s（优化 66%）**
- **部署与工程优化**：部署于 NVIDIA GB10 服务器，突破长音频处理限制（支持 120 秒 +），以 API 封装服务；**会议处理时长缩短 30%**

**项目成果**：落地多说话人会议语音理解核心功能，实现语音内容精准对齐与结构化解析；支持离线运行、可复现可扩展；完整实验报告（RAG 提升 31%，GRPO 相关性 0.82）+ 技术文档（5 份）+ 可运行代码（~650 行核心代码）

**技术栈**：Python, PyTorch, Transformers, ChromaDB, SentenceTransformers, BGE, FunASR, Fun-Audio-Chat, Qwen3-30B, CosyVoice, GRPO
