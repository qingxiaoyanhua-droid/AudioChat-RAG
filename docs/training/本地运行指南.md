# 本地运行指南

> 无需服务器，在本地 macOS/Windows 即可运行和演示

---

## 方案对比

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| **完整方案** | 完整功能 | 需要 GPU，模型大 (16GB+) | 正式运行 |
| **简化方案** | 仅需 CPU，模型小 (2GB) | 无语音合成，仅文本 | 演示/开发 |
| **纯 RAG 方案** | 最快启动 (5 分钟) | 仅 RAG 检索功能 | 功能验证 |

## 方案 1: 简化方案（推荐，仅需 CPU）

### 步骤 1: 创建环境

```bash
cd /Users/setupmac/DiTX-Clerk

# 创建虚拟环境
python3 -m venv .venv
source .venv/bin/activate

# 安装依赖
pip install torch torchaudio
pip install transformers soundfile
pip install chromadb sentence-transformers scikit-learn
pip install funasr modelscope
```

### 步骤 2: 下载模型（仅 2GB）

```bash
# 下载 FunASR 模型
modelscope download --model iic/speech_paraformer-large-vad-punc_asr_nat-zh-cn-16k-common-vocab8404-pytorch --local_dir ./models/funasr
```

### 步骤 3: 测试 RAG 功能

```bash
source .venv/bin/activate

python -c "
from audiochat.rag.storage import MeetingMemoryStore
from audiochat.rag.retriever import AudioChatRetriever

# 初始化
store = MeetingMemoryStore(persist_dir='./rag_storage')

# 添加测试数据
from audiochat.rag.storage import MeetingDocument
docs = [
    MeetingDocument(content='张三：项目进度 80%，预计下周完成', meeting_id='meeting_001', speaker='张三'),
    MeetingDocument(content='李四：前端已完成，等待后端 API', meeting_id='meeting_001', speaker='李四'),
    MeetingDocument(content='王五：测试覆盖率已达 90%', meeting_id='meeting_001', speaker='王五'),
]
store.add_batch(docs)
print('✅ 数据添加成功')

# 测试检索
retriever = AudioChatRetriever(store)
results = retriever.retrieve('项目进度如何？', k=2)
print(f'✅ 检索到 {len(results)} 条结果:')
for r in results:
    print(f'  - {r.content} (分数：{r.relevance_score:.3f})')
"
```

### 步骤 4: 运行 Pipeline（仅 ASR+LLM，无 TTS）

```bash
python scripts/offline_pipeline.py \
  --audio examples/2speakers_example.wav \
  --enable-rag \
  --meeting-id demo_001 \
  --funasr-model ./models/funasr \
  --output-dir output_test
```

## 方案 3: 纯 RAG 演示（最快，5 分钟启动）

### 仅需运行 RAG 检索演示

```bash
# 1. 安装最小依赖
pip install chromadb sentence-transformers

# 2. 运行 RAG 演示
python -c "
from audiochat.rag.storage import MeetingMemoryStore, MeetingDocument
from audiochat.rag.retriever import AudioChatRetriever

# 初始化存储
store = MeetingMemoryStore(persist_dir='./demo_storage')

# 添加模拟会议记录
docs = [
    MeetingDocument(
        content='张三：后端 API 已完成 80%，预计 1 月 15 日完成集成',
        meeting_id='tech_review_2024',
        speaker='张三',
        timestamp='2024-12-15T10:00:00'
    ),
    MeetingDocument(
        content='李四：前端页面完成 60%，等待后端 API 对接',
        meeting_id='tech_review_2024',
        speaker='李四',
        timestamp='2024-12-15T10:05:00'
    ),
    MeetingDocument(
        content='王五：测试用例已编写 50 个，覆盖率目标 85%',
        meeting_id='tech_review_2024',
        speaker='王五',
        timestamp='2024-12-15T10:10:00'
    ),
]
store.add_batch(docs)
print('✅ 添加 3 条会议记录')

# 检索演示
retriever = AudioChatRetriever(store)

queries = [
    '后端进度如何？',
    '前端完成情况',
    '测试覆盖率目标',
]

for query in queries:
    print(f'\n🔍 查询：{query}')
    results = retriever.retrieve(query, k=1)
    if results:
        print(f'  结果：{results[0].content}')
        print(f'  分数：{results[0].relevance_score:.3f}')
        print(f'  说话人：{results[0].speaker}')
"

# 清理演示数据
rm -rf ./demo_storage
```

## 面试演示脚本（1 分钟 Demo）

创建 `demo_quick.py`:

```python
"""
1 分钟面试演示脚本
展示 RAG 检索和 GRPO 评估功能
"""

from audiochat.rag.storage import MeetingMemoryStore, MeetingDocument
from audiochat.rag.retriever import AudioChatRetriever
from evaluation.grpo_eval import GRPOEvaluator

print("=" * 50)
print("AudioChat-RAG 演示")
print("=" * 50)

# 1. RAG 检索演示
print("\n[1/3] RAG 检索演示...")

store = MeetingMemoryStore(persist_dir='./demo_storage')

# 添加模拟数据
docs = [
    MeetingDocument("张三：项目进度 80%，下周完成", "m1", "张三"),
    MeetingDocument("李四：前端完成，等后端 API", "m1", "李四"),
    MeetingDocument("王五：测试覆盖率 90%", "m1", "王五"),
]
store.add_batch(docs)

# 检索演示
retriever = AudioChatRetriever(store)
results = retriever.retrieve("项目进度如何？", k=2)

print(f"  ✅ 检索到 {len(results)} 条结果")
for r in results:
    print(f"     - {r.content} (说话人：{r.speaker})")

# 2. GRPO 评估演示
print("\n[2/3] GRPO 评估演示...")

evaluator = GRPOEvaluator()

# 模拟评估
from evaluation.grpo_eval import GRPOEvalResult
result = GRPOEvalResult(
    cer_score=0.75,
    fluency_score=0.85,
    relevance_score=0.80,
    overall_score=0.80
)

print(f"  CER 分数：{result.cer_score:.2f}")
print(f"  流畅度：{result.fluency_score:.2f}")
print(f"  相关性：{result.relevance_score:.2f}")
print(f"  综合得分：{result.overall_score:.2f}")

# 3. 统计信息
print("\n[3/3] 系统统计...")
stats = store.get_stats()
print(f"  存储文档数：{stats['total_documents']}")

# 清理
import shutil
from pathlib import Path
shutil.rmtree('./demo_storage', ignore_errors=True)

print("\n" + "=" * 50)
print("✅ 演示完成！")
print("=" * 50)
```

运行演示：

```bash
python demo_quick.py
```

## 常见问题

### Q1: modelscope 下载失败

```bash
# 使用镜像
export MODELSCOPE_CACHE="./models"
pip install modelscope -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q2: chromadb 导入失败

```bash
# 重新安装
pip uninstall chromadb
pip install chromadb==0.4.24
```

### Q3: 内存不足

```bash
# 使用小模型
pip install sentence-transformers
# 使用 bge-small-zh 替代 bge-large-zh
```

### Q4: macOS M 系列芯片

```bash
# 使用 MPS 加速
export PYTORCH_ENABLE_MPS_FALLBACK=1
pip install torch torchvision torchaudio
```
